# Blur & Sharpen

## Sharpen

**Parameter**: `sharpen`  
**Units**: Boolean  
**Default**: false

Sharpens the image.

Examples:
- `?sharpen=false` (or not defined) - original image
- `?sharpen=true` - sharpened image

## Blur

**Parameter**: `blur`  
**Units**: Number  
**Range**: 0-100  
**Default**: 0

Applies a blur effect to the image. The higher the value, the stronger the blur effect.

Examples:
- `?blur=0` - original image
- `?blur=10` - slight blur
- `?blur=25` - moderate blur
- `?blur=60` - strong blur
